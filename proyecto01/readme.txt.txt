Para instalar las dependencias del proyecto
Instalación de dependencias: 
Cuando alguien (o tú mismo) necesite instalar 
las dependencias del proyecto, puede ejecutar 
el siguiente comando desde la raíz del proyecto:
1- activar entorno virtual
2- pip install -r requirements.txt(crea o actualiza el archivo con las dependencias)
3- levantar servidor desarrollo: python manage.py runserver
4- ingresar a la url por defecto http://127.0.0.1:8000/  o a la que corresponda segun settings.py
